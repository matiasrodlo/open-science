# Science Downloader for macOS

Download research papers with ease on macOS.

## Installation

### Method 1: Applications Folder (Recommended)
1. Copy "Science Downloader.app" to your Applications folder
2. Launch from Applications folder
3. Grant necessary permissions when prompted

### Method 2: Direct Launch
1. Right-click "Science Downloader.app"
2. Click "Allow" for Science Downloader
3. Double-click to launch

## Usage

### Starting the Application
1. Double-click "Science Downloader.app"
2. Wait for the application to start
3. Your default browser will open automatically
4. The application runs on http://localhost:5000

### Stopping the Application
- Close the browser tab
- Or press Ctrl+C in the terminal window

## Data Storage

### Application Data
The application stores data in:
~/Library/Application Support/Science Downloader/

### Downloaded Papers
Papers are saved to the location you specify in the web interface.

## Troubleshooting

### Permission Issues
If you get permission errors:
1. Go to System Preferences > Security & Privacy
2. Click "Allow" for Science Downloader
3. Try launching again

### Port Issues
If port 5000 is busy, the application will automatically find an available port.

### Support
For issues or questions, please refer to the main documentation.

## System Requirements

- macOS 10.13 (High Sierra) or later
- Internet connection
